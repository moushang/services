# 服务器方案

## 方案一 最低配

```
  ECS系列2(1核1GB 1M带宽) 
= 816元
```

`缺点:带宽慢`

`优点:固定收费,费用低`

## 方案二 

```
   ECS系列2(1核1GB 1M带宽) 816元/年
+ 对象存储OSS资源包(包月) 40G 5元*x12月=60元/年
+ 对象存储OSS资源包(包月) 下午流量包60元/半年*2=120元/年
= 996
```

`优点:OSS放图片资源便宜,速度快`




## 方案三

```
负载均衡SLB(按量付费)  0.02 /小时 * 0.80 /GB
+ 谋商服务器(1核2GB)  600/年
```

`缺点:流量大时费用高`

`优点:SLB分担流量的费用,收费低,按量收费`
